## What is claudeCode?

`claudeCode` is an AI CLI agent that can autonomously read, edit, test, and commit code. Under the hood it combines:

- A **Query Engine** that drives multi-turn conversation loops against an LLM
- A rich **tool ecosystem** (file editing, shell execution, web search, notebook editing, computer vision)
- A **Permission & Security System** with fine-grained rule matching and an Auto-Mode Classifier
- **Git Integration** for diff-aware, context-rich coding assistance
- **Teammates** — background sub-agents that run tasks in parallel
- **Model Context Protocol (MCP)** for plugging in external tool servers
- Advanced modes: Buddy System, Ultraplan, Auto-Dream, Remote Bridge, and Computer Use
