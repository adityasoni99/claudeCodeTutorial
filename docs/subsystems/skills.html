Tutorial: skills

This project manages skills, which are custom capabilities or commands (like %%INLINE0%% or %%INLINE1%%) available to the AI model. It supports two main types of skills: bundled skills built directly into the CLI binary, and user-defined skills loaded dynamically from markdown files in the filesystem. The system handles discovery, parsing, and the dynamic generation of context-aware prompts that guide the model's behavior.
